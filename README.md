# РТ - Выставочное приложение

Electron-приложение для выставочного стенда с сенсорным интерфейсом.

## Особенности

- **Разрешение**: 1920x1080
- **Режим**: Fullscreen, без рамок (kiosk mode)
- **Интерфейс**: Адаптирован для сенсорного экрана
- **Технологии**: Electron + React + TypeScript + Vite

## Функциональность

1. При запуске автоматически воспроизводится приветственное видео
2. По клику на видео открывается главное меню
3. Главное меню содержит 4 большие кнопки для навигации

## Установка зависимостей

```powershell
npm install
```

## Запуск в режиме разработки

```powershell
npm run electron:dev
```

Команда запустит:
- Vite dev server на порту 5173
- Electron приложение с hot reload

## Сборка приложения

```powershell
npm run electron:build
```

Скомпилированное приложение будет в папке `dist-electron`.

## Структура проекта

```
PT/
├── electron/          # Главный процесс Electron
│   ├── main.js       # Основной файл приложения
│   └── preload.js    # Preload скрипт
├── src/              # React приложение
│   ├── App.tsx       # Главный компонент
│   ├── App.css       # Стили
│   ├── main.tsx      # Точка входа
│   └── index.css     # Глобальные стили
├── public/           # Статические файлы
│   └── video/        # Видеофайлы
├── index.html        # HTML шаблон
├── package.json      # Зависимости
├── vite.config.ts    # Конфигурация Vite
└── tsconfig.json     # Конфигурация TypeScript
```

## Добавление видео

Поместите видеофайл `intro.mp4` в папку `public/video/`.

## Примечания

- Приложение настроено на запуск в режиме киоска (fullscreen без возможности выхода)
- Для выхода из режима разработки используйте Alt+F4
- DevTools открываются автоматически в режиме разработки
